Architecture Recommendation

Instead of 7 department-level Projects, use ~18 functional Projects grouped by workflow cluster. Many departments have use cases with very different context needs (e.g., Marketing's "Offer Creation" needs a pricing database while "SEO Workflow" needs web search). Splitting by function gives each Project a focused instruction set and relevant knowledge files. V2 adds 3 new use cases: Auto Sales Follow-Up, Outbound Enquiry Generation (both Marketing), and AI Travel Desk (Servicing).
